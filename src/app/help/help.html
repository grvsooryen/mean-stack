Prerequisites

Before installing this package you need certain tools installed on your system

  • NodeJs
  • Bower
  • Grunt-cli
  • MongoDB

Installation

Follow these set of command to begin with installation.

git clone https://github.com/grvsooryen/mean-stack.git
cd mean-stack
npm install
bower install
mongo
    > use contactlist
    > exit
grunt
node server.js
To update changes please you grunt watch
